= Требования лицензии

* Для работы с модулем {wc} лицензия на систему {dv} должна содержать дополнительную опцию _{dv} Thin client_. В дополнительной опции указано максимальное количество пользовательских подключений к {wc}у.
// * В случаях, когда {wc} и сервер {dv} будут установлены на отдельных компьютерах, лицензия на систему {dv} должна содержать дополнительную опцию _{dv} Concurrent Servers_ с количеством соединений 2 или более.
* Для работы с программой {kvr} лицензия на систему {dv} должна содержать дополнительную опцию: _{dv} Конструктор карточек_.
* Для работы с функцией агрегации потребуется дополнительная опция лицензии: _Docsvision Модуль Интеллектуальный поиск_. См. подробнее в разделе xref:admin:grid-aggregate.adoc[].
* Для xref:user:grid-ribbon.adoc#citations[работы с цитатами] в гриде в режиме ленты также потребуется дополнительная опция лицензии: _Docsvision Модуль Интеллектуальный поиск_. В противном случае в режиме ленты цитаты не будут отображаться.
