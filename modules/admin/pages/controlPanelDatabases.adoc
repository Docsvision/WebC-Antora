= Раздел "Базы данных"

== Настройки подключения к серверу

Раздел _Базы данных_ открывает страницу настроек подключения {wc} к серверу {dv}.

.Страница настроек "Базы данных"
image::controlPanel_databases.png[Страница настроек "Базы данных"]

.Страница настроек "Базы данных"
[cols="33,66",frame="none",grid="none"]
|===
|`Адрес сервера`
|Полный или сокращенный адрес сервера платформы {dv}, к которому подключается {wc}.

|image:buttons/bt_save.png[save]
|Сохраняет адрес сервера после изменения.

|_Тенанты_
|Список подключенных к {wc} баз данных указанного сервера {dv}.
|===

TIP: Если нужная БД в списке отсутствует, её можно добавить через _{cns}_ при помощи Мастера баз данных _Руководство Администратора модуля Платформа_, пункт "Базы данных").

[NOTE]
====
После добавления новой БД необходимо её настроить. Потребуется проверить локализации, импортировать решения и настроить виды карточек по умолчанию. Подробнее см. п. xref:setingsExtended.adoc[Расширенная настройка].
====

== Изменение адреса подключения к серверу

.Чтобы изменить адрес подключения к серверу {dv}:
. Введите в поле `Адрес сервера` адрес сервера {dv}.
+
include::partial$rest-soap-1.adoc[]
+
include::partial$rest-soap-2.adoc[]
+
. Нажмите на кнопку image:buttons/bt_save.png[Сохранить].
+
Если был указан адрес формата SOAP, он будет сокращен до `\http://{Адрес сервера {dv}}/{dv}/`. Если в конфигурационном файле {wc} xref:changeTransport.adoc[указан транспорт COM], сокращение адреса не выполняется.

Пользователи Модуля могут работать с несколькими БД. Текущей активной считается БД, псевдоним которой указан в адресной строке до первой точки, например: `\http://{dv}DB.company/{dv}WebClient/#/Dashboard`.
