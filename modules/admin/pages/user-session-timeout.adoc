= Изменить время ожидания пользовательской сессии

// tag::webconfig[]
Администратор может изменить максимальное время ожидания ответа (таймаут) от сервера {dv}. Время ожидания устанавливается в миллисекундах.

.Чтобы изменить таймаут пользовательской сессии:
. Откройте конфигурационный файл `{webconfig}`.
. Перейдите к секции {cnf-sgrps} > System].
. Измените значение параметра `ConnectionTimeout`.
+
[source,json]
----
  "Docsvision": {
    "WebClient": {
      "SettingGroups": {
        "System": {
          "ConnectionTimeout": 3600000 <.>
        }
      }
    }
  }
----
<.> Время ожидания пользовательской сессии равное 1 часу. Значение задаётся в миллисекундах.
