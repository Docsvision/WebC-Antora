= Настройки кэширования разметок

{wc} использует серверный кэш для хранения подготовленных разметок карточек, основного меню, дашборда и др.

Для каждой локации разметок (см. xref:layouts:locations-about-settings.adoc[Руководство пользователя программы {kvr}]) используется собственный кэш, для которого может быть установлен требуемый режим кэширования:

include::partial$table.adoc[]
|*_Disabled_*
|Кэширование отключено.

|*_PerUser_*
|Для каждого пользователя используется собственный кэш.

|*_AllUsers_*
|Используется общий кэш для всех пользователей.
|===

.Кэширование по умолчанию настроено следующим образом:
[horizontal]
AllUsers::
- Локация "Создание дочернего задания согласования".
- Локация "Панель поиска по столбцам представления".
- Локация "Диалог поисковых параметров".
- Локация "Корень WebFrame".

PerUser::
- Локация "Главная страница".
- Локация "Основное меню".
- Локация "Диалог профиля пользователя".
- Локация "Групповые операции с карточками".

Disabled::
- Остальные локации.

Администратор может изменить режим кэширования. Для выполнения следующих действий нужна программа _{dv} Explorer_ (входит в _{dv} 5 {rk}_).

. Откройте _{dv} Explorer_.
. Подключитесь к серверу и БД {dv}, в которой выполняется изменение.
. В разделе _Cards_ выберите карточку _WebClientSettingsDirectory_.
+
.Раздел "Cards" в {dv} Explorer
image::dv-explorer-webc-dir.png[Раздел "Cards" в {dv} Explorer]
+
. Нажмите кнопку *Card data* на панели команд. Будет открыт редактор данных карточки _WebClientSettingsDirectory_.
. Перейдите к секции _Локации_ карточки.
+
.Данные секции "Локации" карточки "WebClientSettingsDirectory"
image::dv-explorer-webc-dir-locations.png[Данные секции "Локации" карточки "WebClientSettingsDirectory"]
+
. В _Section rows_ выберите строку секции с настройками требуемой локации. Название локации указано в поле _Name_ строки секции.
. Измените значение поля _CacheLevel_ (дважды щелкните по полю для открытия редактора) на требуемый режим кэширования:
* *_0_* -- *_Disabled_*.
* *_1_* -- *_PerUser_*.
* *_2_* -- *_AllUsers_*.
. Закройте _{dv} Explorer_.
. Перезапустите сервер {wc}а.
