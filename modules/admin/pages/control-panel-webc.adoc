{pu} предоставляет базовые настройки протоколирования событий {wc}а. Дополнительные настройки задаются в конфигурационном файле {wc}а (см. xref:log-config.adoc[Настройка журналирования]).

== Изменение параметров размещения
.Чтобы изменить параметры размещения веб-приложения:
- Администратор должен создать новый сайт или выбрать уже созданный.
- Изменить При необходимости название виртуальной папки.
- Нажать на кнопку *Обновить*.
- Веб-приложение {wc}а будет перемещено на выбранный сайт.

Если сайт содержит веб-приложение {wc}а, будет предложено его перезаписать.

Если в IIS отсутствует стандартный пул приложений {wc}а -- _{dv}SitePool_, пул будет создан автоматически.

При создании нового пула необходимо ввести логин и пароль учётной записи. От имени этой учётной записи будет запускаться созданный пул.

См. xref:ROOT:requirements-pool.adoc[Требования к учетной записи пула приложений].
