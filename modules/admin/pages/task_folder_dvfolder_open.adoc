
== Выбор папки

Для выбора папки кликните по её названию в [.dfn .term]_основном меню_ или [.dfn .term]_дашборде_. В зависимости от типа папки, будет открыт список карточек папки, окно xref:ParametricSearch.html[настройки параметрического поиска] или другая страница.

*На уровень выше:* xref:../topics/FolderCommonOperations.html[Общие операции с папками]
