Задание будет отправлено всем выбранным сотрудникам и сотрудникам выбранных организаций, подразделений, групп и ролей. Фактическим исполнителем станет первый сотрудник, взявший задание в работу.
+
[NOTE]
====
Если нужно отправить каждому сотруднику собственное задание, используйте xref:task-groups.adoc[Группу заданий].
====

xref:appendix/ctrlDateTime.adoc[Срок исполнения]::
Плановый срок исполнения задания выбранными исполнителями. По умолчанию устанавливаются текущие дата и время.

xref:appendix/ctrlText.adoc[Примечание]::
Дополнительная информация по заданию.

Поставить на контроль::
Включает функции xref:tasks-user-controller.adoc[контроля исполнения] задания ответственным сотрудником. При включенном контроле станут доступны параметры контроля:

xref:appendix/ctrlStaffDirectoryItems.adoc[Контролёр]:::
Сотрудник, который контролирует выполнение задания.

xref:appendix/ctrlDateTime.adoc[Дата контроля]:::
Дата, когда должен быть выполнен контроль выполнения задания.

xref:tasks-controller-accept.adoc[Требуется приёмка]:::
Включает функцию приёмки задания контролёром. После выполнения задания исполнителем, контролёр может принять результат выполнения или отказать. При включении приёмки будет автоматически включается контроль.
