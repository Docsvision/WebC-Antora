= Отключить папки

include::partial$notInMobileVersion.adoc[]

.Чтобы отключить видимость определённых папок:
. Нажмите на кнопку image:buttons/butt_folder_tree_settings.png[Шестерёнка] в xref:interfaceMainMenu.adoc[Основном меню].
+
****
Будут открыты настройки папок основного меню.
****
+
.Настройка папок основного меню
image::foldertree_folder_settings.png[Настройка папок основного меню]
+
. Выберите в меню пункт _Настроить панель_.
+
****
Слева от названий папок появятся флаги управления подключенными папками.
****
+
.Управление подключенными папками
image::folder_switchoff_dv_folder.png[Управление подключенными папками]
+
. Снимите флаги с папок, которые нужно отключить.
. Нажмите кнопку *Сохранить*, чтобы применить изменения.
+
****
Папки, подключенные пользователем, будут отключены. Стандартные папки будут скрыты.
****
