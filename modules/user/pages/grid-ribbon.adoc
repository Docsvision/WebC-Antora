= Лента

При нажатии правой кнопки image:buttons/grid-ribbon.png[Таблица-список] на панели управления переключается режим отображения грида.
При переключении на режим отображения "лента" окно позиционируется на самой верхней карточке в окне.

.Общий вид ленты
image::grid-ribbon-view.png[Общий вид ленты]

Как и в табличном режиме отображения, каждая строка ленты представляет отдельную карточку.

== Карточки

В левой части строки отображается индикатор прочитанности карточки. Если в строке текст выделяется жирным и слева строки отображается синий индикатор, значит, карточка не прочитана. Чтобы прочитать карточку, нажмите на индикатор один раз. Чтобы отметить карточку как непрочитанную, нажмите слева от заголовка карточки.

В каждой строке отображается флаг для выбора карточки и активации xref:grid-batch.adoc[групповых операций].

Слева на панели инструментов отображается флаг для выбора всех карточек в папке и xref:grid.adoc#count[количество карточек].

В центре панели инструментов отображается кнопка переключения режима отображения грида: image:buttons/grid-ribbon.png[Таблица-список].

== Сортировка

Также на панели инструментов отображается значок сортировки image:buttons/add-sort.png[Три полоски]. По нажатию на значок появляется выпадающий список, в котором можно задать направление сортировки.

При смене сортировки данные в ленте меняют порядок.

== Настройки

Окно "Настройки ленты" в целом аналогично xref:grid-toolbar.adoc[настройкам таблицы], за исключением возможности переноса строк.

Настройка _Колонки_ в режиме ленты имеет свои особенности. Информация, которая отображается в колонках в режиме таблицы, представлена в виде строк карточки.

.Настроенные "колонки" в ленте
image::ribbon-settings.png[Настроенные "колонки" в ленте]

По умолчанию для отображения выбираются три первых колонки, например _Рег. номер_, _Дата регистрации_, _Название_.

Все колонки добавляются в карточку в ленте друг под другом. Чтобы изменить их порядок, поменяйте расположение заголовков.

Настройки колонок для ленты не переносятся в табличное отображение. Если в ленте что-то поменяли, в том же представлении грида настройки остаются неизменными.

.В правой части окна настроек ленты располагаются настройки ширины колонки:
* При выборе пункта *Минимальная* все данные ленты будут отображены в несколько узких колонок (значение по умолчанию).
+
.Минимальная ширина колонки
image::small-ribbon.png[Минимальная ширина колонки]
+
* При выборе пунктов *Средняя* и *Максимальная* данные ленты будут отображены в несколько широких колонок, а поля данных будут распределяться по области ленты более равномерно.
+
.Максимальная ширина колонки
image::big-ribbon.png[Максимальная ширина колонки]

Текст, который не помещается в одну строку, переносится всегда.

== Экспорт

При экспорте ленты в Excel результирующий файл всё равно отображается в виде таблицы, но с теми полями и в том порядке, что были выбраны в окне _Настройки ленты_.

== Прокрутка

Прокрутка в ленте работает аналогично прокрутке в xref:grid.adoc#scroll[таблице].
