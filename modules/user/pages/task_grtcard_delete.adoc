
== Удаление карточки Группа заданий

Карточка _Группа заданий_ может быть удалена пользователем при выполнении следующих условий:

* карточка задания должна быть разблокирована;
* карточка должна быть в состоянии `Подготовка`;
* пользователь должен являться автором группы заданий.

См. общую инструкцию по удалению в пункте xref:cardsDelete.adoc[Удаление карточки]. При удалении группы заданий ссылки на неё будут удалены из других карточек.
