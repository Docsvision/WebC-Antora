= Согласование договора

include::partial$excerpts.adoc[tags=incomingFolder]
+
. Откройте полученное задание с описанием _Согласование документов_ в состоянии _Не начато_.
. Выполните действия с согласуемыми документами:
+
* xref:approvalFiles.adoc[Ознакомьтесь с составом согласуемых файлов].
* Ознакомьтесь с договором: нажмите на ссылку в разделе *Ссылка на карточку* для перехода к договору.
* Если требуется, xref:approvalFiles.adoc#addVersion[добавьте собственную версию договора].
+
. Чтобы принять решение по согласованию, нажмите кнопку:
+
* *Согласовать* или *Согласовать с замечаниями* на xref:cards-terms.adoc#cardsOperations[панели операций]. После того как все согласующие завершат свои задания, документ будет передан на xref:contractsApprovedConsolidation.adoc[консолидацию].
* *Не согласовано*. Документ будет отправлен на новый цикл согласования.
* *Делегировать*, *В работу*.

Общая информация о принятии решения по согласованию приведена в пункте xref:approval-performer.adoc[].
