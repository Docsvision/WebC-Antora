= Файловая панель

Для работы с файлами карточек _Задание_ и _Группа заданий_ предназначен элемент управления `_Файловая панель_`.

.ЭУ `_Файловая панель_` предоставляет следующие функции работы с файлами:
- <<addFiles,Добавлять файлы>>,
- <<deleteFiles,Откреплять файлы>>,
- <<previewFiles,Просматривать файлы>> (ограничено),
- <<editFiles,Редактировать файлы>> (ограничено),
- <<newVersionOfFile,Загружать новые версии файлов>>,
- <<openCard,Просматривать файлы документов "Задания" или "Группы заданий">>.

Интерфейс элемента зависит от режима работы с карточкой и наличия загруженного файла:

* Элемент управления без добавленного файла при создании новой карточки:
+
.Файловая панель при создании карточки Задание
image::taskFilePanelEmpty.png[Файловая панель при создании карточки Задание]

* Элемент управления при просмотре карточки:
+
."Файловая панель" при просмотре карточки "Задание"
image::taskFilePanelEmptyRead.png["Файловая панель" при просмотре карточки "Задание"]

* Элемент управления с добавленными файлами при просмотре карточки:
+
."Файловая панель" с добавленными файлами в карточке "Задание"
image::taskFilePanelView.png["Файловая панель" с добавленными файлами в карточке "Задание"]

[#addFiles]
== Добавление файлов

.Добавить файл можно двумя способами:
* Перетащите файл из файловой системы в область элемента `_Файловая панель_`.
* Когда файл не добавлен, нажмите на элемент и выберите файл с помощью Проводника.
+
При добавлении файла будет создана новая карточка _Файл_ с выбранными файлом.

.Условия добавления файлов:

* Типы файлов, которые можно добавлять: `.docx`, `.pdf`, `.rtf`, `.dot`, `.doc`, `.dotm`, `.docm`, `.dotx`, `.ods`, `.odt`, `.csv`, `.xlt`, `.xls`, `.xltm`, `.xlsm`, `.xltx`, `.xlsx`, `.jpg`, `.jpeg`, `.gif`, `.png`, `.bmp`, `.tiff`, `.tif`, `.emf`, `.wmf`, `.xslt`, `.xsl`, `.xml`, `.htm`, `.html`, `.txt`, `.log`, `.cs`, `.asmx`, `.ascx`, `.aspx`, `.js`, `.cpp`, `.h`, `.css`, `.idl`, `.ini`, `.java`, `.rc`, `.rc2`, `.shtm`, `.shtml`, `.sql`, `.zip`, `.rar`, `.7z`, `.vsd`, `.vsdx`, `.msg`, `.one`, `.pptx`, `.ppt`, `.avi`, `.wav`.

TIP: Для возможности загрузки файлов .ascx, .cs и .java требуется xref:admin:installAfter.adoc#java[дополнительная настройка].

[#deleteFiles]
== Открепление файлов

.Чтобы открепить файл:
* При создании или редактировании карточки нажмите кнопку image:buttons/redCross.png[Крестик] в строке файла.
* При просмотре карточки нажмите image:buttons/verticalDots.png[Три вертикальные точки] *> Открепить* в строке файла.
+
****
Карточка _Файл_ при этом не удаляется.
****

[#editFiles]
== Редактирование файла

Чтобы открыть файл в связанной программе в режиме просмотра карточки нажмите image:buttons/verticalDots.png[Три вертикальные точки] *> Открыть* в строке файла. Открытый для редактирования файл будет заблокирован от изменений со стороны других пользователей. Элемент управления поддерживает версионинг, это значит, что при сохранении файла с изменениями будет создана новая версия.
+
.Условия редактирования файлов:
****
Данная возможность доступна только для файлов Microsoft Office (соответствующие программы должны быть установлены): `.doc`, `.docx`, `.docm`, `.xls`, `.xlsx`, `.xlsm`, `.ppt`, `.pptx`, `.odt` и `.ods`.

До завершения редактирования файла не закрывайте карточку, из которой он был открыт.

[#previewFiles]
== Предварительный просмотр

Щелкните по названию файла, чтобы открыть его в режиме предварительного просмотра. Инструкция по работе с компонентом предпросмотра приведена в пункте xref:appendix/ctrlFilePreview.adoc[Предпросмотр файла].

== Скачивание файла

Чтобы скачать файл на устройство, нажмите image:buttons/verticalDots.png[Три вертикальные точки] *> Скачать*.

[#newVersionOfFile]
== Загрузка новой версии файла

.Чтобы добавить новую версию существующего файла:
. Нажмите image:buttons/verticalDots.png[Три вертикальные точки] *> Загрузить версию*.
. Выберите новую версию файла с устройства. Если название файла новой версии не совпадает с названием существующей, версия будет переименована. Для переименования потребуются права на переименование файла.

[#openCard]
== Открытие карточки файла

Для открытия карточки _Файл_, содержащей файл _Задания_ или _Группы задания_, нажмите image:buttons/verticalDots.png[Три вертикальные точки] *> Открыть карточку файла*.
