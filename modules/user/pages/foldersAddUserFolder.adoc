= Подключить пользовательские папки

include::partial$notInMobileVersion.adoc[]

Пользователь может добавить в дерево папок необходимые папки, на которые у него есть права доступа.

== Подключение пользовательских папок

.Чтобы подключить пользовательские папки:
. Нажмите на кнопку image:buttons/butt_folder_tree_settings.png[Шестерёнка] в xref:interfaceMainMenu.adoc[Основном меню].
+
****
Будут открыты настройки папок основного меню.
****
+
.Настройка папок основного меню
image::foldertree_folder_settings.png[Настройка папок основного меню]
. Выберите в меню пункт _Присоединить папку_.
+
****
Появится окно со списком доступных папок.
****
+
.Выбор из доступных папок
image::folder_select.png[Выбор из доступных папок]
+
. Выберите одну или несколько папок, которые нужно подключить.
+
TIP: Для выбора нескольких папок удерживайте кнопку kbd:[Ctrl] или kbd:[Shift].
. Нажмите кнопку *Выбрать*.
+
****
Выбранные папки будут добавлены в раздел _МОИ ПАПКИ_ xref:interfaceStandardFoldersTree.adoc[дерева папок].
****

== Видимость дочерних папок

Видимость дочерних папок зависит от типа подключенной родительской папки:

* Дочерние папки стандартных (значок image:buttons/icon_standard.png[Папка]) и виртуальных (значок image:buttons/icon_virtual.png[Папка с лупой посередине]) папок всегда отображаются.
* Дочерние папки папки-делегата (значок image:buttons/icon_delegate.png[Папка со стрелкой]) отображаются, если в настройках папки включен режим _Показывать подпапки_.

== Отображение внешней страницы в папке

Если подключаемая папка настроена на открытие внешней страницы вместо представления (см. _Руководство пользователя Windows-клиента_), должен выполняться ряд условий:

. Если папка настроена на открытие отчета службы Microsoft SQL Server *2016* Reporting Services, в URL-адрес отчета в настройках папки нужно добавить параметр `_rs:Embed=true_`. Например, `\http://CompanyReportServer.com/Reports/DocumentsReport?rs:Embed=true`.
+
Если используется Microsoft SQL Server с версией ниже 2016, добавлять указанный параметр не нужно.
+
NOTE: Для доступа к отчету используются данные пользователя, от имени которого запущен браузер. А не того, от чьего имени выполнен вход в {wc}.
+
. Сервер SQL Server Reporting Services и сервер {wc} должны находиться в одном домене.
+
WARNING: Подключаемая страница должна разрешать встраивание. Открытие папки со страницей, встраивание которой запрещено, завершится с ошибкой `Refused to display 'Site' in a frame because it set 'X-Frame-Options' to 'DENY'`.
+
. После изменения URL-адреса в настройках папки нужно перезапустить IIS и повторно зайти в {wc}.

NOTE: Если подключенная папка содержала подпапки-делегаты, которые были удалены, нужно отключить и повторно подключить папку, чтобы обновить структуру папок.
