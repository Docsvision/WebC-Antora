== Однострочный режим

По умолчанию данные в представлении отображаются в однострочном режиме. В данном режиме скрывается часть данных, выходящая за границы столбца представления. Чтобы посмотреть содержимое полностью, наведите курсор мыши на обрезанный текст.

.Отображение данных в однострочном режиме
image::viewarea_docsvision_folder_dots.png[Отображение данных в однострочном режиме]

== Многострочный режим

Пользователь может включить многострочный режим с переносом строк. В данном режиме данные отображаются полностью.

.Отображение данных в многострочном режиме
image::viewarea_size.png[Отображение данных в многострочном режиме]

Чтобы переключиться в многострочный режим, нажмите кнопку image:buttons/butt_grid_two_lines.png[Кривая линия] на панели инструментов представления. Чтобы вернуться к однострочному режиму, нажмите кнопку image:buttons/butt_grid_one_line.png[Прямая линия].

При следующем открытии данной папки будет использоваться установленный способ переноса текста.
