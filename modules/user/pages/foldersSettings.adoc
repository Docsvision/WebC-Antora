= Изменить настройки папки

include::partial$notInMobileVersion.adoc[]

Пользователь может изменить настройки папки, если данная возможность разрешена при настройке {wc}а.

Настройки стандартных папок не могут быть изменены. Используйте xref:foldersAddUserFolder.adoc[пользовательские папки].

.Чтобы изменить настройки папки:
. В xref:interfaceMainMenu.adoc[основном меню] наведите курсор мыши на папку, настройки которой изменяются.
+
****
Будет показана кнопка image:buttons/verticalDots.png[Три вертикальные точки].

Кнопка скрыта, если функция недоступна пользователю.
****
+
. Нажмите image:buttons/verticalDots.png[Три вертикальные точки] > Настройки папки.
+
Будет открыта панель настроек папки.
+
.Панель настроек папки
image::configFolderConfigDialog.png[Панель настроек папки]
+
TIP: При отсутствии прав на изменение папки, панель настроек будет открыта в режиме чтения (без возможности изменения настроек).
+
. Измените требуемые настройки папки: `Имя папки`, `Тип папки` (см. xref:foldersCreateNew.adoc[Создание новых папок]), xref:views.adoc[представление] по умолчанию и необходимость xref:foldersNewCardNotification.adoc[подсветки непрочитанных] карточек.
. Нажмите кнопку *Права доступа*, если нужно изменить xref:foldersSecurity.adoc[настройки безопасности] папки.
. Нажмите кнопку *ОК* для сохранения изменений.
