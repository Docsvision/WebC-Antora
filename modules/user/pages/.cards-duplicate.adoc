= Дублировать карточку

Для карточки или группы карточек можно создать дубликаты.

[#context]
== Дублирование карточки из контекстного меню

.Чтобы создать дубликат карточки:
. Откройте любую папку в представлении {wc}а.
. Вызовите команду дублирования любым из способов:
+
. Откройте контекстное меню, нажав правой кнопкой на строке карточки.
+
. Выберите пункт _Дублировать карточку_.
+
.Дублировать карточку
image::duplicate-card.png[Дублировать карточку]
+
. В зависимости от выбранного значения свойства "xref:layouts:ctrl/system/cardManagement.adoc#window[Окно выбора папки]" может отображаться окно выбора папки.
+
Если в конфигурационном файле {wc} по адресу {webconfig} параметр "xref:admin:.duplicate-settings.adoc[Показывать окно выбора папки при дублировании]" указан в значении `*Показывать*`, открывается окно выбора папки, в которую будет добавлен ярлык карточки. +
Если параметр установлен в значении `*Не показывать*`, карточка создается в системе без папки, созданный дубликат карточки открывается в новой вкладке в разметке создания.

== Дублирование карточки из управления карточкой

. Откройте карточку в режиме просмотра и выберите в строке с названием карточки значок дублирования image:buttons/duplicate.png[Дублировать карточку].
. Доступность значка в строке с названием карточки регулируется свойством `*Дублирование*` элемента управления "xref:layouts:ctrl/system/cardManagement.adoc[Управление карточкой]". +
. В зависимости от выбранного значения свойства "xref:layouts:ctrl/system/cardManagement.adoc#window[Окно выбора папки]" может отображаться окно выбора папки.

== Дубликат для группы карточек

Описание групповой операции _Дублировать карточки_ приведено в разделе "xref:grid-batch.adoc#duplicate[Дублировать карточки]".

[#details]
== Особенности создания дубликатов:

* При создании дубликата создаётся карточка того же вида и с теми же настройками, что и исходная карточка.
* Дубликат карточки будет иметь только те поля, для которых задано свойство _Копирование_. В дубликат копируются как статические поля, так и расширенные.
* Если невозможно выполнить копирование в связи с ограничением прав доступа, будет выведена ошибка `Невозможно выполнить данную операцию в связи с ограничением прав доступа.`
* Карточка создается в начальном состоянии.
