= Создание группы заданий

. xref:cardsCreateNew.adoc[Создайте карточку] вида menu:Группа заданий[Группа заданий УД]. 
+
****
Будет открыто окно новой карточки.
****
+
.Новая группа заданий вида "Группа заданий УД"
image::grtcard_create.png[Новая группа заданий вида "Группа заданий УД"]
+
. Заполните поля группы заданий.
+
. Нажмите кнопку *Отправить на исполнение*, чтобы отправить задания исполнителям или *Сохранить*, чтобы сохранить группы заданий с возможностью отправки. Карточка будет открыта в режиме просмотра.
+
****
Задания для исполнителей формируются в течении некоторого времени.
****

xref:SimpleFields.adoc[Длительность]::
Длительность исполнения заданий относительно текущей даты и времени.
+
При изменении длительности, срок исполнения будет пересчитан с учётом бизнес-календаря (см. правила <<calendar,выше>>).

Последовательное исполнение/Параллельное исполнение:::
Тип исполнения заданий группы:
* *_Параллельное исполнение_*. Задания могут выполняться одновременно всеми исполнителями;
* *_Последовательное исполнение_*. Задания будут выполняться исполнителями последовательно по отношению к порядку исполнителей в списке.
+
При изменении типа исполнения нужно подтвердить пересчет даты исполнения заданий.

Исполнители::
Исполнители группы заданий с индивидуальными настройками исполнения. См. пункт xref:taskGroupsUsersPerformers.adoc[Выбор исполнителей группы заданий].
+
IMPORTANT: Обязательно для ввода. Должен быть добавлен хотя бы один исполнитель.

Поставить на контроль::
Включает функцию xref:tasksUserController.adoc[контроля исполнения] заданий группы ответственным сотрудником. При включенном контроле станут доступны параметры:

xref:StaffDirectoryItems.adoc[Контролёр]::: Сотрудник, который контролирует выполнение заданий.

xref:DateTime.adoc[Дата контроля]::: Дата, когда должен быть выполнен контроль выполнения заданий.

Требуется приёмка::
Включает функцию xref:tasksControllerAcceptance.adoc[приёмки контролёром] выполненных заданий. При включении приёмки будет автоматически включен контроль.
+
[NOTE]
====
Ознакомиться с особенностями контроля и приёмки можно в пункте xref:taskGroupsControlSpecifics.adoc[].
====
