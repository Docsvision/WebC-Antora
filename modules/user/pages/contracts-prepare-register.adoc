= Подготовка и регистрация договора

. xref:cards-new.adoc[Создайте карточку] вида menu:Документ[Документ УД > Договор].
+
.Карточка документа "Договор"
image::contract-create-mode.png[Карточка документа "Договор"]
+
. Заполните договор.
+
Основное::
xref:appendix/numerator.adoc[Регистрационный номер]:::
Регистрационный номер договора (нажмите кнопку image:buttons/number.png[Получить номер] для получения номера). Если номер не присвоен, это будет сделано автоматически при сохранении карточки.
xref:appendix/date-time.adoc[Дата регистрации]:::
Дата регистрации договора. По умолчанию устанавливается текущая дата.
xref:appendix/directory-designer-row.adoc[Категория договора]:::
Категория к которой относится договор. Значение выбирается из узла "Группы договоров" Конструктора справочников.
xref:appendix/simple-fields.adoc[Предмет договора]:::
Описание предмета договора.
Контрагент::
xref:appendix/partner.adoc[Контактное лицо]:::
Ответственный сотрудник контрагента.
xref:appendix/partner-company.adoc[Организация]:::
Контрагент, с которым заключён договор.
Условия::
Сумма договора:::
Стоимость договора в цифровом выражении.
Валюта -- валюта договора из списка:::
По умолчанию выбираются "рубли" (RUB).
xref:appendix/date-time.adoc[Действует с, по]:::
Период действия договора.

С нашей стороны::
xref:appendix/staff-department.adoc[Организация]:::
организация или подразделение, ответственное за договор.
xref:appendix/staff-directory-items.adoc[Ответственный]:::
сотрудник, который отвечает за заключение и подписание договора с контрагентом. По умолчанию выбирается текущий сотрудник.
xref:appendix/staff-directory-items.adoc[Подготовил]:::
сотрудник, который регистрирует договор. По умолчанию выбирается текущий сотрудник.
xref:appendix/staff-directory-items.adoc[Подписывает]:::
сотрудник, который отвечает за подписание договора в организации. Данному сотруднику будет передан полученный от контрагента согласованный вариант договора.
xref:appendix/files.adoc[Файлы]::
Файлы договора. По умолчанию к карточке договора прикладывается шаблон договора, настроенный администратором. Если требуется, удалите данный файл и приложите новый.
. Нажмите кнопку *Создать*. После сохранения договор будет открыт в режиме просмотра.
