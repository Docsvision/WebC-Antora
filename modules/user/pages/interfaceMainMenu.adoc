По умолчанию основное меню скрыто. Нажмите кнопку image:buttons/folderTree.png[Меню], чтобы временно открыть его. Чтобы закрепить панель в открытом состоянии, нажмите кнопку image:buttons/folderTreePin.png[Закрепить]. Для возврата к режиму временного отображения нажмите кнопку image:buttons/folderTreeUnfix.png[Уголок влево] или image:buttons/folderTree.png[Меню].

[NOTE]
====
При настройке {wc}а может быть включено альтернативное поведение основного меню. При этом кнопки закрепления/открепления скрываются, а меню автоматически закрепляется/открепляется при нажатии image:buttons/folderTree.png[Меню].
====

В мобильной версии {wc}а недоступные кнопки меню скрыты. Вместо недоступных добавляются кнопки открытия xref:interfaceUserMenu.adoc[Меню пользователя] и _Справки_ по работе с программой, а также перехода к полной версии {wc}а.
