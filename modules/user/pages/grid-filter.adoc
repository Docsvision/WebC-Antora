[#filter]
== Работа с фильтрацией

.Чтобы применить фильтр в представлении:
. Наведите курсор на заголовок колонки.
+
В режиме планшета нажмите один раз на заголовок колонки.
+
. Нажмите кнопку image:buttons/filter-black.png[Фильтр].
+
.Кнопка фильтр в заголовке таблицы
image::grid-filter.png[Кнопка фильтр в заголовке таблицы]
+
. Выберите нужный фильтр.
+
Пользователь может найти нужный фильтр, начав вводить текст в поисковой строке. Совпадения подсвечиваются цветом во всех строках списка.
+
.Поиск фильтров
image::filter-search.png[Поиск фильтров]
+
. Чтобы выбрать нужное значение, установите напротив него флаг. Выбрать можно одно или несколько значений.
+
Если значений много (больше 15), они загружаются поэтапно: сначала первые 15, затем при прокрутке частями загружаются остальные. Значения отображаются в порядке получения с сервера.
+
Длинные названия фильтров обрезаются по ширине окна. При наведении на такое название появляется всплывающая подсказка с полным текстом.
+
. Можно комбинировать несколько фильтров в разных колонках. Чтобы использовать несколько фильтров, примените фильтр к колонке, не сбрасывая предыдущие.
+
[NOTE]
====
Если фильтр применён, в колонке отображается красный значок фильтра:

.В колонке применён фильтр по названию
image:filtered.png[В колонке применён фильтр по названию]
====
+
. Чтобы изменить фильтр в колонке, нажмите на него ещё раз и выберите новое значение.
. Чтобы сбросить фильтр, нажмите на кнопку image:buttons/recycle-bin-red.png[Красная урна]. Чтобы сбросить все фильтры, нажмите на кнопку *Показать все*.

[#period]
== Фильтрация по датам

Если нажать кнопку image:buttons/filter-black.png[Фильтр] в колонке с датой, появится окно, позволяющее выбрать два варианта фильтрации: вкладки <<date,Дата>> и <<log,Журнал>>.

.Использование фильтра по дате
image::filter-date.png[Использование фильтра по дате]

Нельзя одновременно использовать фильтры из обеих вкладок. Например, вы выбрали фильтр на вкладке _Дата_ и нажали кнопку *Применить*, затем перешли на вкладку _Журнал_ и нажали кнопку *Применить*. В этом случае будет применён только фильтр с вкладки _Журнал_, фильтр на вкладке _Дата_ будет сброшен.

[#date]
=== Вкладка "Дата"

В фильтре на вкладке _Дата_ значение даты всегда отображается без времени, даже если в столбце время указано.

Вкладка _Дата_ отображается первой при выборе фильтра.

При выборе данной вкладки открывается календарь, который позволяет выбрать конкретную дату, либо диапазон дат.

.Выбор периода
image:select-range.png[Выбор периода]

.По умолчанию в календаре выбрано сегодняшнее число.
* Чтобы выбрать другую дату, нажмите на число в календаре.
* Чтобы выбрать другой год, нажмите на год отображаемый
* Чтобы выбрать диапазон дат, выберите два числа. Выберите дату начала периода, нажмите на неё, затем выберите дату окончания периода и нажмите на неё. Нажмите кнопку *Применить*, чтобы применить фильтр.
+
NOTE: Если вы выбрали дату окончания периода в прошлом (например, начало в 2020 году, а окончание в 2010), датой начала будет выбрана более ранняя дата (2010).
+
* Чтобы выбрать сегодняшнюю дату, нажмите на иконку календаря.
+
.Выбранный период и кнопка "Сегодня"
image::selected-range.png[Выбранный период и кнопка "Сегодня"]

[#log]
=== Вкладка "Журнал"

.Вкладка "Журнал
image::date-log.png[Вкладка "Журнал]

На вкладке _Журнал_ отображается список уникальных дат, встречающихся в текущей папке. Список дат выводится в порядке, который получен с сервера. Если список слишком длинный, появляется вертикальная прокрутка.

Чтобы выбрать значение, установите напротив него флаг и нажмите кнопку *Применить*
