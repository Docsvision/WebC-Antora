= Добавление отчёта по заданию

Отчёт о выполнении всех содержащихся в задании работ может быть добавлен при завершении задания или в процессе работы с ним. Отчётом может являться: текстовый комментарий, файл, ссылка на карточку.

Добавление отчёта разрешено всем _Исполнителям_ (текущему исполнителю, делегатам всех уровней), _Ответственному исполнителю_ (если есть такая роль) во всех состояниях, кроме `Подготовка`, `Завершено`, `Отклонено`, `Отозвано`.

При прикреплении отчёта тип ссылки всегда будет определяться настройкой, заданной для соответствующего вида задания в _Справочнике видов карточек_ системы {dv} (вкладка _Задания_, поле `Типы ссылок для отчётов`).

.Чтобы добавить отчёт в процессе выполнения задания:

. Откройте карточку задания.
. Перейдите к разделу _Отчёт_.
+
.Раздел "отчёт" в карточке Задания
image::task-reports.png[Раздел "отчёт" в карточке Задания]
+
. Приложите отчёт:
- Файл отчёта.
- Существующую или новую карточку.
- Текстовый отчёт.

. Нажмите image:buttons/plus.png[Плюс] > image:buttons/attach.png[Скрепка].
. Выберите добавляемый файл.

.Отчёт в виде новой карточки:
. Нажмите *image:buttons/plus.png[Плюс]* > *image:buttons/frame-plus.png[Прямоугольник с плюсом в круге]*.
. Выберите вид документа, заполните поля новой карточки и сохраните изменения. Ссылка на связанную карточку документа появится в разделе _Отчёт_.

.Отчёт в виде существующей карточки:
. Нажмите *image:buttons/plus.png[Плюс]* > *image:buttons/greay-frame.png[Прямоугольник со стрелкой]*.
. Выберите существующую карточку документа.

.Текстовый отчёт:
. Щелкните по области *Добавить текстовый отчёт либо файл отчёта*.

. Введите комментарий в появившемся поле для ввода текста.
. Нажмите кнопку *Сохранить*, чтобы добавить текст в отчёт.
