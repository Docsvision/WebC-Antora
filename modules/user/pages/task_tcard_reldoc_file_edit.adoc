
== Открытие файла связанного документа из карточки Задание

. [.ph .cmd]#Откройте карточку задания.#
. [.ph .cmd]#Перейдите к разделу "Документы".#
. [.ph .cmd]#В строке открываемого файла нажмите [.ph .menucascade]#*image:buttons/verticalDots.png[Три вертикальные точки]* > *Открыть*#. Файл будет открыт в редакторе файлов данного типа (см. условия в пункте xref:Files.adoc[Список файлов]): в режиме чтения или редактирования, если есть права на редактирования. При открытии в режиме редактирования файл будет заблокирован.#
