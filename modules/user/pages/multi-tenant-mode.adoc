= Работа в мультитенантном режиме

{wc} поддерживает работу в мультитенантном режиме без дополнительных настроек. В мультитенантном режиме имеет значение адрес открытия {wc}а.

До первой точки указывается псевдоним БД, в которой планируется работа, например, `\http://**oldDB.**company.com/Docsvision/WebClient/#/Dashboard`.

Подробнее о мультитенантном режиме см. в разделе xref:admin:multi-tenant-mode.adoc[документации] по администрированию {wc}а.
