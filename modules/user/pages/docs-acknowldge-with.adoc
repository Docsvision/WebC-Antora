= Ознакомиться с ходом исполнения связанных заданий и групп заданий

Ознакомиться с ходом исполнения задания или группы заданий можно с помощью раздела _Ход исполнения_.

. Откройте карточку документа в xref:cards-open-modes.adoc#read-mode[режиме чтения].
. Перейдите к разделу _Ход исполнения_.
+
Для раскрытия групп заданий щелкните по значку image:buttons/open-group.png[Треугольник вершиной вниз] слева от значка группы.
+
Чтобы ознакомиться с ходом исполнения в виде xref:appendix/task-tree.adoc[дерева исполнения], нажмите кнопку *Дерево исполнения*. Для просмотра хода исполнения в виде xref:appendix/task-table.adoc[таблицы] нажмите кнопку *Таблица исполнения*.
