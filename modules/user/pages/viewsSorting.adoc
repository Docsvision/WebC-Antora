****

[NOTE]
====
В {wc}е не работает сортировка по столбцу с вычисляемым полем, в котором используется функция конкатенации.

Из-за существующего ограничения MicrosoftSQL возможно обрезание части данных в строках столбцов, по которым выполнена сортировка.

Данные могут обрезаться, если суммарное количество данных в этих столбцах превысит существующее ограничение в 4000 символов для текстовых данных. Значение может отличаться для других типов данных.
====
