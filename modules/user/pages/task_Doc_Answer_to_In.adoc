
== Создание исходящего документа в ответ на входящий

Чтобы создать исходящий документ в ответ на входящий, выполните следующие действия:

. xref:OpenCard.adoc[Откройте карточку] вида _Входящий_.
. На xref:CardOperations.adoc[панели операций] нажмите кнопку *Создать исходящий*. Будет открыта новая карточка исходящего документа. Часть данных карточки будет заполнена данными из входящего документа.
. Заполните поля карточки и сохраните изменения.
+
Созданная карточка будет добавлена в список ссылок в раздел *Ссылки*.
