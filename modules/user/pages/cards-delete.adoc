= Удаление карточки

. Откройте карточку в xref:cards-open-modes.adoc#read-mode[режиме чтения].
. Нажмите кнопку image:buttons/recycle-bin-simple-black.png[Урна] в строке с названием карточки.
+
****
Будет предложено подтвердить удаление.
****
+
. Нажмите *ОК* для подтверждения.
+
****
После удаления будет открыта страница xref:interface-dashboard.adoc[Дашборд].
****
+
