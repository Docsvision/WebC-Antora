= Удалить папку

include::partial$notInMobileVersion.adoc[]

Пользователь может удалить папку, если данная возможность разрешена при настройке {wc}а.

.Чтобы удалить папку:
. В xref:interfaceMainMenu.adoc[Основном меню] наведите курсор мыши на удаляемую папку.
+
****
Будет показана кнопка image:buttons/verticalDots.png[Три вертикальные точки].

Кнопка скрыта, если функция недоступна пользователю.
****
+
. Нажмите image:buttons/verticalDots.png[Три вертикальные точки] *> Удалить папку*.
+
****
Будет показан диалог подтверждения удаления.
****
+
NOTE: Если отсутствуют права на удаление папки, при выборе появится предупреждение.
+
. Подтвердите удаление папки.
+
****
Папка будет удалена со всеми подпапкам. В зависимости от xref:admin:deletePermanently.adoc[настроек] {wc}а папка удаляется в Корзину или окончательно.
****
