= Настройка разметок

Настройка разметки -- формирование набора настроенных элементов управления, необходимых для работы пользователя с данными. Разметки настраиваются с помощью xref:dl_ui_layouteditor.adoc[Редактора разметок].

.Стандартные этапы настройки разметки:
. Сформируйте общую структуру разметки с помощью xref:dl_layoutElements.adoc[элементов разметки].
. Сформируйте отдельные блоки для работы с данными при помощи элементов разметки _Блок_, _Страница вкладок_, _Таблица_.
. Добавьте в разметку элементы управления для отображения или ввода данных.
. Настройте элементы управления. Свяжите ЭУ с данными, настройте общие свойства, обработчики событий, локализации и т.д.

=== Локализация разметок

{kvr} предоставляет возможность локализовать разметки, определять для различных текстовых переменных значения в зависимости от языка интерфейса {wc}а.
