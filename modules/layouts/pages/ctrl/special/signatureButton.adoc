= Кнопка электронной подписи

Элемент управления позволяет подписывать карточки без файлов (выполнять подписание атрибутов).

WARNING: ЭУ можно разместить только в разметке карточки типа "Документ". Для разметок других типов ЭУ в библиотеке не отображается.

=== Данные

==== Операция редактирования Настроек подписи

Свойство регулирует доступность кнопки для вызова окна "Настройки подписи", в котором осуществляется настройка списка сертификатов.

==== Подписывать без метки

Флаг определяет, будет ли отображаться выпадающий список способов подписания. Если флаг установлен, будет выполняться подписание без метки и выпадающий список не будет отображаться.
