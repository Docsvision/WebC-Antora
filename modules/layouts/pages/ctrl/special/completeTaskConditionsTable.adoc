= Параметры завершения задания

Элемент управления `_Параметры завершения задания_` предназначен для отображения дополнительных параметров завершения задания, настроенных в Справочнике видов карточек и в индивидуальных настройках задания.

Элемент управления может быть добавлен только в разметку локации menu:WebFrame[Разметки для элементов управления > Диалоги операций карточек > Завершить].
