==== Скачать файлы с подписью

Когда свойство активно, разрешено скачивание файлов с подписью.

include::partial$properties.adoc[tags=stdCss]

==== Формат подписи

Когда свойство `*Скачать файлы с подписью*` активно, подпись будет скачиваться в выбранном из списка формате: *_.p7s_*, *_.sgn_*, *_.sig_*, *_.sign_*.
