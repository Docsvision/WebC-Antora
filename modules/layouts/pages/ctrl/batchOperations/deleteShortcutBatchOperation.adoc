= Групповые операции / Удаление ярлыков

Элемент управления `_Групповые операции / Удаление ярлыков_` вызывает операцию удаления ярлыков группы выбранных карточек из текущей папки.

TIP: Данный ЭУ может быть добавлен только в разметку локации menu:WebFrame[Компоненты папки > Групповые операции с карточками].
