= Состояние выделения карточек папки

Элемент управления `_Состояние выделения карточек папки_` предназначен для отображения количества карточек, выбранных в режиме групповых операций.

ЭУ может быть добавлен в разметку локации _Групповые операции с карточками_ и в корневой элемент для ЭУ групповых операций (`_BatchOperationsNode_`).
