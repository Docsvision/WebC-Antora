= Групповая операция / Делегирование

Элемент управления `_Групповая операция / Делегирование_` вызывает операцию делегирования для группы выбранных карточек Задание.

ЭУ может быть добавлен в разметку локации _Групповые операции с карточками_ и в корневой элемент для ЭУ групповых операций (`_BatchOperationsNode_`).
