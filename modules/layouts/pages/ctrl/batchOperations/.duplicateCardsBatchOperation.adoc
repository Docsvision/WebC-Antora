= Групповая операция / Дублировать карточки

Элемент управления `_Групповая операция / Дублировать карточки_` вызывает операцию дублирования для группы выбранных карточек.

ЭУ может быть добавлен в разметку локации _Групповые операции с карточками_ и в корневой элемент для ЭУ групповых операций (`_BatchOperationsNode_`).
