= Групповая операция / Перемещение ярлыков

Элемент управления `_Групповая операция / Переместить ярлык_` вызывает операцию перемещения ярлыков группы выбранных карточек из текущей папки в выбранную. Перед выполнением операции будет представлена возможность выбора папки, в которую перемещаются ярлыки.
