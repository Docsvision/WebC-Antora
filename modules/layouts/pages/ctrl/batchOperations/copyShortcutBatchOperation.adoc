= Групповая операция / Копирование ярлыков

Элемент управления `_Групповая операция / Копирование ярлыков_` вызывает операцию копирования ярлыков группы выбранных карточек из текущей папки в выбранную. Перед выполнением операции будет представлена возможность выбора папки, в которую копируются ярлыки.

TIP: Данный ЭУ может быть добавлен только в разметку локации menu:WebFrame[Компоненты папки > Групповые операции с карточками].
