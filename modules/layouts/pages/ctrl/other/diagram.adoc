= Диаграмма

Элемент управления добавляет отображение диаграмм на главной странице. Позволяет изменять вид и содержание диаграмм.

=== Внешний вид

==== Высота

Высота блока с диаграммой. Значение указывается в пикселях.

Чтобы высота устанавливалась автоматически, укажите значение `0`.

==== Заголовок

Текст, отображаемый над диаграммой.

`*Локализуемый*`

==== Показывать легенду

Свойство включает или отключает отображение легенды диаграммы.

==== Положение легенды

Позволяет выбрать местоположение легенды. Свойство доступно только, если флаг `*Показывать легенду*` установлен.

.Доступные значения:
* Снизу
* Сверху
* Слева
* Справа

==== Цветовая схема

Определяет преобладающий цвет диаграммы. Для выбора доступны 9 цветов и разноцветная схема.

==== Цветовой режим

Определяет цветовой режим диаграммы: *_Одноцветный_* -- будет использоваться один цвет или *_Монохромный_* -- будут использоваться оттенки выбранного цвета.

=== Поведение

==== 1-й столбец

Свойство определяет первый столбец из списка доступных в представлении для указанной папки или указанного представления.

==== 2-й столбец

Свойство определяет второй столбец из списка доступных в представлении для указанной папки или указанного представления.

Свойство может быть не указано для других видов диаграмм.

Свойство не доступно для типа диаграмм: гистограмма и кольцевая диаграмма.

==== ID папки

Позволяет задать папку, из которой будут получены данные для диаграммы.

Свойство не доступно, если указан `*Поисковый запрос*`. Чтобы указать *`ID папки*` нужно очистить `*Поисковый запрос*` -- указать значение *_<Не задано>_*.

==== ID представления

Позволяет указать представление, которое будет служить источником данных для диаграммы.

Свойство доступно вне зависимости от других установленных свойств.

==== Перейти в папку

Когда флаг установлен, нажатие на сектор диаграммы переводит в папку, указанную в свойстве `*ID папки*`. В открывшейся папке автоматически применяется фильтр, соответствующий выбранному сектору диаграммы.

==== Поисковый запрос

Позволяет выбрать один из предварительно настроенных поисковых запросов.

Свойство не доступно, если в `*ID папки*` указана папка. Чтобы снова выбрать `*ID папки*`, очистите поисковый запрос -- выберите значение *_<Не задано>_*.

include::partial$properties.adoc[tags=stdCss]

==== Тип диаграммы

Выберите из раскрывающегося списка тип диаграммы, которая должна отображаться.

.На выбор доступны 4 типа диаграмм:
* *_Гистограмма_*.
* *_Гистограмма с накоплением_*.
* *_Диаграмма с областями с накоплением_*.
* *_Кольцевая_*.
