= Вкладки

Элемент разметки `_Вкладки_` предназначен для размещения нескольких переключаемых страниц. На каждой из страниц может быть размещён набор отображаемых или изменяемых данных карточки.

Каждая вкладка ЭР `_Вкладки_` представляет собой отдельный элемент разметки -- xref:ctrl/layoutElements/tabPage.adoc["Страница вкладок"]. В ЭР `_Вкладки_` может быть добавлено необходимое количество ЭУ `_Страница вкладок_` со своими наборами подчиненных ЭУ. В ЭУ `_Вкладки_` могут быть добавлены только ЭУ `_Страница вкладок_`.

ЭР `_Вкладки_` является контейнером, который непосредственно не отображается в карточке.

==== Основной таб в мобильной верстке

При работе на мобильном устройстве, если в разметке используется несколько ЭР `_Вкладки_`, один из ЭР `_Вкладки_` может быть назначен основным. Страницы основного ЭР `_Вкладки_` отображаются под панелью управления. Страницы прочих ЭР `_Вкладки_` -- на месте, определенном разметкой данной карточки.

.Элемент разметки "Вкладки" в карточке, открытой на мобильном устройстве
image::tabSampleMobile.png[Элемент разметки "Вкладки" в карточке, открытой на мобильном устройстве]

Данное свойство определяет, какой из нескольких ЭР `_Вкладки_` является основным:

* Когда флаг установлен, текущий ЭР `_Вкладки_` является основным.
* Когда флаг не установлен, текущий ЭР `_Вкладки_` будет размещен в карточке стандартным образом (согласно настройкам разметки). В одной разметке может быть только один основной ЭР `_Вкладки_`. Данная настройка не влияет на разметку карточки при работе на персональном компьютере.
