= Контейнер заголовка

Элемент управления `_Контейнер заголовка_` является контейнером для элементов управления, которые должны отображаться в верхней части окна {wc}а.

.В контейнер могут быть добавлены только следующие элементы управления:
* `_xref:ctrl/webFrame/webFrameMainMenuButton.adoc[]_`
* `_xref:ctrl/webFrame/webFrameCompanyLogo.adoc[]_`
* `_xref:ctrl/webFrame/webFrameSearchPanel.adoc[]_`
* `_xref:ctrl/webFrame/webFrameDirectorySearchPanel.adoc[]_`
* `_xref:ctrl/webFrame/webFrameUserPanel.adoc[]_`
* `_xref:ctrl/webFrame/webFrameHelpButton.adoc[]_`
* `_xref:ctrl/webFrame/webFrameCurrentPageName.adoc[]_`
* `_xref:ctrl/standard/button.adoc[]_`
* `_xref:ctrl/standard/textBox.adoc[]_`
* `_xref:ctrl/standard/number.adoc[]_`
* `_xref:ctrl/standard/url.adoc[]_`
* `_xref:ctrl/standard/dropdown.adoc[]_`
* `_xref:ctrl/standard/label.adoc[]_`
* `_xref:ctrl/standard/radioGroup.adoc[]_`
