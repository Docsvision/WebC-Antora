= Html тег

Элемент управления `_Html тег_` предназначен для добавления в разметку произвольного HTML-тега с заданными содержимым.

.Пример отображения произвольного содержимого с помощью элемента управления "Html тег"
image::html-tag.png[Пример отображения произвольного содержимого с помощью элемента управления "Html тег"]
