= Карточки папки

include::partial$inDevelopment.adoc[]

Элемент управления `_Карточки папки_` предназначен для отображения содержимого папки с применением выбранного представления.

.Пример элемента управления "Карточки папки"
image::grid.png[Пример элемента управления "Карточки папки"]

== Свойства

include::partial$properties.adoc[tags=general]

=== Внешний вид

[#ribbon]
==== Использовать ленту по умолчанию

Когда флаг активен, по умолчанию в представлении вместо грида будет отображаться xref:user:ribbon.adoc[лента].
include::partial$excerpts.adoc[tags=separate]

[#transparency]
==== Непрозрачность

Свойство регулирует прозрачность грида. Прозрачность может быть настроена в диапазоне от `0` до `100` (в процентах), где `0` означает максимальную прозрачность, а `100` -- максимальную непрозрачность. Значение по умолчанию -- `100`.
