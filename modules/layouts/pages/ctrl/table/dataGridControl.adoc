= Контрол табличных данных

Элемент управления отображает данные в виде таблицы на основании модели полученной с сервера, сформированной программно.

Таблицу можно использовать как в карточке, так и в любой разметке, например в разметке дашборд.

=== Данные

==== Название плагина

Обязательный параметр, на основании которого происходит поиск плагина для формирования таблицы.

TIP: Создание плагина подробнее описано в xref:programmer:client/data-grid-control.adoc[руководстве программиста].

==== Параметры формирования таблицы

Набор дополнительных атрибутов, которые можно указать в Конструкторе Web-разметок. Указанные атрибуты попадут в плагин.
