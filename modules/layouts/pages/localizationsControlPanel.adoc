= Панель управления ресурсами локализации

_Панель управления ресурсами локализации_ -- инструмент управления локализованными ресурсами. При помощи панели управления можно создавать и изменять текстовые ресурсы, которые могут быть использованы в разметках для настройки элементов управления: текстовых меток, подсказок и т. п.

.Панель управления ресурсами локализации
image::uiLocalizationsTable.png[Панель управления ресурсами локализации]

_Панель управления ресурсами локализации_ может быть открыта двумя способами:

* из xref:layoutsEditor.adoc[редактора разметок] -- для этого нажмите кнопку image:buttons/dots.png[Три точки] в настраиваемом свойстве элемента управления с поддержкой локализации.
* вызвана из меню программы {kvr} -- команда Файл > Локализация.
+
В данном случае в _Таблице локализаций_ будут отображаться все ресурсы xref:solutionChangeCurrent.adoc[текущего выбранного Решения].

Если требуется настроить локализацию для языка, отсутствующего в _Панели управления ресурсами локализации_, добавьте его с помощью _Панели управления {wc}а_ и разрешите к использованию. Подробности см. пункт xref:admin:controlPanelLanguages.adoc#add-new[Добавление нового языка].

== Компоненты панели управления

.Компоненты панели управления ресурсами локализации:
include::admin:partial$table.adoc[]
|*Добавить*, *Удалить* |Добавляют и удаляют ресурсы.
|Фильтр ресурсов |Фильтрует ресурсы по принадлежности к текущей разметке. Доступен, если _Панель управления ресурсами локализации_ открыта из разметки. Фильтр содержит варианты:

Все разметки. Отображаются все ресурсы Решения, в которое входит текущая разметка.
Текущая разметка. Отображаются только ресурсы, использованные в текущей разметке.
|_Таблица локализаций_ |Таблица, содержащая значения ресурсов для поддерживаемых языков.

|*ОК*, *Отмена* |Кнопки сохранения и отмены сохранения.
|===

[TIP]
====
Шрифт строк в _Таблице локализаций_ указывает на "используемость" локализованного ресурса:

* Стандартный шрифт -- локализованный ресурс используется в одной разметке (возможно в нескольких элементах управления).
* Жирный шрифт -- локализованный ресурс используется в нескольких разметках.
* Серый цвет шрифта -- локализованный ресурс не используется.
====
