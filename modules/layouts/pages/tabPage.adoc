
== Страница вкладок

Элемент разметки *Страница вкладок* представляет собой одну вкладку в _ЭР_ xref:tab.adoc[Вкладки], на которой размещаются требуемые элементу управления.

image::controls_tab_sample.png[Пример элемента разметки "Страница вкладок"]

[NOTE]
====
[.note__title]#Важное замечание:# ЭР *Страница вкладок* может быть встроен только в _ЭР_ *Вкладки*.
====
